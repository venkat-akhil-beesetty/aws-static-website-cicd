AWS Static Website Deployment with CI/CD

A DevOps project demonstrating automated deployment of a static website using AWS S3, CloudFront, IAM and GitHub Actions.

Project Overview

This project demonstrates how to host a static website on AWS using Amazon S3 and CloudFront while implementing a CI/CD pipeline using GitHub Actions. Every code change pushed to GitHub is automatically deployed to AWS.

CI/CD Workflow

  1. Developer pushes code to GitHub.
  2. GitHub Actions workflow is triggered.
  3. AWS credentials are loaded securely.
  4. Files are synchronized to Amazon S3.
  5. CloudFront cache invalidation is performed.
  6. Updated content becomes available globally.
